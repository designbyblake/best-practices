<!DOCTYPE html>

A document type declaration, or DOCTYPE, is an instruction that associates a particular SGML or XML document (for example, a webpage) with a document type definition (DTD) (for example, the formal definition of a particular version of HTML).

<!DOCTYPE html>

  • The HTML5 Doctype
  • Introduces new HTML5 elements
  • Introduces data-attributes
  • New input types, Placeholder text, autofocus, required & more
  • Is used to validate HTML

<headings>

H1 — H6

  • Headings create the document outline
  • Only one H1 per page, the HTML5 document outline algorithm does not work
  • Never choose a heading for size or style, use CSS
  • Used by search engines for SEO

<headings>

H1 — H6

  • Never skip a level when going down, never go from h2 to h4
  • Can skip levels when going up
  • You can have multiple headings of the same level in a row

Images

  • Images should always have an alt attribute.
  • Alt attributes should describe the purpose of the image not what it is.
    • Bad Alt = "Guy holding a bat"
    • Good Alt = "Men enjoying America's favorite past time, Baseball!"
    • If the image is decorative only have a blank alt attribute
    • If no alt attribute screen readers read the image path
    • Alt attributes are used by Google for image search

Paragraphs

  • Always wrap paragraphs in the proper element, <p>
  • Never use two <br><br> to make 2 paragraphs, the spacing will not be correct.
  • Paragraphs can contain images, links, spans & elments such as <em>, <small> and <abbr>
